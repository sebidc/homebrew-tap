Homebrew tap for `sebidc` formulae.

## Install

Tap the repository:

```sh
brew tap sebidc/tap
```

Install a formula:

```sh
brew install appldl
brew install check-installs-tui
```

## Formulae

- `appldl`
- `check-installs-tui`
